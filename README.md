# Airdrop NFT API

Demo to mint NFT contracts on Polygon L2 `matic` and `mumbai` testnet.

## Install 

```
npm i
```

## Setup

```
PORT=5001

# Basic Auth
USERNAME=username
PASSWORD=password

# Account 1
PRIVATE_KEY=PRIVATE_KEY_WITH_0x

# EIP1559 priority
PRIORITY_MULTIPLIER_FEE=fast
```

## Test

```
npm run test
```

## Run

```
npm run start
```
